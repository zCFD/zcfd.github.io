zCFD Tutorials
==============

The following sections are step-by-step guides to running zCFD on different test cases. All are freely available to download.

Getting zCFD
------------

The first step for any of the cases is to download and install zCFD from the `zCFD website <https://zcfd.zenotech.com/>`_. Once you have the tarball simply un-tar it:

.. code-block:: shell

    $ tar -zxvf zCFD-icc-sse-impi-2019.01.68-Linux-64bit.tar.gz

Once zCFD has been installed the zCFD environment needs to be activated:

.. code-block:: shell

    $ source /path/to/zCFD/bin/activated
    (zCFD)$
